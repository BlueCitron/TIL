# 오늘 배운 것을 정리하기 위한 Repository
 - 오늘 배운 것들을 기술별로 폴더를 생성해 정리
 - 오늘 배운 것들은 설명 가능 또는 코딩 가능하면 공부 완료
 - 하나의 주제에 대해 하나의 md 파일을 생성하여 정리
